ANGSD biSNP discovery — C. gariepinus 226-sample hatchery cohort
SNP discovery on per-chromosome chunks of fClaHyb_Gar_LG.fa (28 pseudochromosomes, 963.9 Mb). Computed from BAMs filtered through MODULE_1 QC (depth, mapping rate, t-len). Per-chromosome chunked array runs (S06_run_bisnp_discovery_chunks.slurm), output as .mafs.gz for downstream BEAGLE imputation + PCA.
Parameters (MODULE_2A canonical, applied uniformly across all 28 chromosomes):
-GL 1SAMtools genotype-likelihood model — standard for low-coverage WGS
-minQ 25Minimum base quality
-minMapQ 25Minimum mapping quality
-baq 1BAQ recalibration
-C 50Excessive-mismatch downweight
-setMinDepthInd 3Min per-individual depth (from MODULE_1 QC distribution)
-setMaxDepthInd 57Max per-individual depth (from MODULE_1 QC distribution)
-minInd 200Min individuals with data per site (200/226 = 88%)
-SNP_pval 1e-6SNP significance threshold
-minMaf 0.05Minimum minor allele frequency
-pestFolded SFS prior (no outgroup; F₁ hybrid hatchery violates HWE — folded by design)
For other cohorts, the precomp export script (planned, schema 2.15 §8.x) emits these parameters as a top-level angsd_bisnp_params object so this panel can read them generically. This particular tooltip text is hard-coded for the gariepinus project's known canonical run.
